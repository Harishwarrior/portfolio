# Harish's Portfolio

Personal portfolio website showcasing my work as a Flutter Developer, including mobile apps, packages, tools, and technical articles.

## Tech Stack

- **Framework**: Next.js 15
- **UI Library**: React 19
- **Styling**: Tailwind CSS v4
- **Animations**: Motion & Motion-Primitives
- **Content**: MDX for blog posts

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Project Structure

- `app/` - Main application pages and components
- `app/data.ts` - Portfolio data (projects, work experience, blog posts)
- `app/header.tsx` - Header component with personal introduction
- `components/ui/` - Reusable UI components
- `public/carousel/` - Personal photos for the carousel

## Customization

Edit `app/data.ts` to update projects, work experience, and social links.

Edit `app/header.tsx` to modify the personal introduction.

## Deployment

Deploy to Vercel, Netlify, or any platform that supports Next.js.
